Introducción a GitHub

GitHub es una plataforma donde puedes guardar tus proyectos de programación y compartirlos con otras personas. Permite trabajar en equipo, hacer cambios en el código y mantener versiones organizadas. Se utiliza principalmente para:

  • Guardar proyectos en la nube
  • Compartir código
  • Crear portafolios profesionales
  • Trabajar en equipo
